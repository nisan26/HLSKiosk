# HLSKiosk - בנייה אוטומטית של APK ב-GitHub

הקוד הזה מובנה אוטומטית ל-APK בעננים של GitHub - לא צריך להתקין Android Studio במחשב שלך!

## איך להשתמש - מדריך מהיר

### 1️⃣ צור חשבון GitHub (אם אין)
- לך ל-https://github.com/signup
- הירשם (חינמי)

### 2️⃣ צור מאגר חדש (Repository)
1. אחרי הרשמה - לחץ על **"+"** למעלה משמאל → **New repository**
2. שם: `HLSKiosk` (או כל שם אחר)
3. **Public** (חינמי - או Private אם יש לך תשלום)
4. **אל תסמן** "Add a README" - אנחנו מעלים קבצים קיימים
5. לחץ **Create repository**

### 3️⃣ העלה את הקבצים
1. בדף החדש שנפתח - לחץ על **"uploading an existing file"** (קישור באמצע)
2. גרור לתוך הדפדפן את **כל הקבצים** של הפרויקט (או לחץ "choose your files")
3. בתחתית - לחץ **Commit changes**

### 4️⃣ המתן לבנייה אוטומטית
1. לחץ על הלשונית **Actions** למעלה
2. תראה "Build APK" רץ עם עיגול צהוב מסתובב
3. המתן **3-5 דקות** עד שזה הופך לעיגול ירוק ✅

### 5️⃣ הורד את ה-APK
**אפשרות A** (מומלצת) - דרך Releases:
1. לחץ על **Releases** בעמודה הימנית של דף הראשי
2. תראה "HLSKiosk Build #1" - לחץ עליו
3. בתחתית, תחת "Assets" - לחץ על **HLSKiosk.apk** והוא יורד למחשב

**אפשרות B** - דרך Actions:
1. לחץ על **Actions** → על הבנייה האחרונה (הירוקה)
2. גלול למטה לחלק **Artifacts**
3. לחץ על **HLSKiosk-APK** - יורד ZIP, חלץ אותו - בפנים יש את ה-APK

### 6️⃣ התקן ב-X96Q דרך אפליקציית המנהל
1. הפעל את `run.bat` של אפליקציית המנהל
2. לחץ **🔗 התחבר** (IP: 192.168.1.183)
3. לחץ **📂** ובחר את `HLSKiosk.apk` שהורדת
4. לחץ **⬇️ התקן / עדכן APK**
5. כשנגמר - לחץ **▶️ הפעל** והקופסה תתחיל לשדר!

## עדכוני קוד עתידיים

כשתרצה לשנות משהו בקוד:
1. ערוך את הקובץ ישירות ב-GitHub (לחץ עליו → אייקון העיפרון)
2. GitHub יבנה APK חדש אוטומטית תוך 5 דקות
3. הורד את הגרסה החדשה ושלח לקופסה דרך המנהל

## תכונות האפליקציה

✅ עולה אוטומטית בכל הדלקת קופסה
✅ מנגנת `http://satview.ddns.net/lus300/index.m3u8` כברירת מחדל
✅ ניתן לשנות URL מרחוק דרך אפליקציית המנהל
✅ Reconnect אוטומטי אם הזרם נופל
✅ Watchdog שמחיה את האפליקציה אם היא קורסת
✅ buffer של 30-60 שניות ליציבות
✅ עובד טוב על X96Q Pro עם 1GB RAM
